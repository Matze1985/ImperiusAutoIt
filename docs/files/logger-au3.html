_LogLevel

Func _LogLevel($oSelf,
$iLevel)

Sets the log level.  Looger supports 6 logging levels: TRACE, DEBUG , INFO, WARNING , ERROR and FATAL.  The default logging level is INFO.  This means that only log entries with the logging level INFO and higher are output.

Parameters

$iLevelInt.  The log level.  $LOG_TRACE = 0, $LOG_DEBUG = 1 $LOG_INFO = 2, $LOG_WARN = 3, $LOG_ERROR = 4, $LOG_FATAL = 5.
